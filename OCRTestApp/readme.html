TesseractOCR.app Readme

System Requirements

  • Mac OS X 10.5 or later (10.6 recommended)
  • Universal binary
  • 64 bit support for Intel

Usage

Drag an image to the left hand panel and the recognised text (if any) will appear in the right hand panel

Tiff images work best because they have least conversion. Use high resolution images if you can (ideally more than 150dpi).

Low resolution images perform worse.

If you have serious need of OCR this may not be the program for you.

Tesseract OCR Library

This application contains tesseract as a framework. The source code is downloadable from the web page at

http://www.malcolmhardie.com/ocr/
